# ポートフォリオサイト リニューアル（2027年3月リニューアル予定）

## 概要

このリポジトリは、ポートフォリオサイトのリニューアル用です。

全面的な再設計・再構築は、**2027年3月** に完成予定です。  
その時期には、以下の状態になっていることを想定しています。

- 開発経験の増加
- 掲載できる制作物の充実
- キャリア方向性の明確化
- フロントエンド技術力の向上
- デザイン判断力の成長

それまでは、現行ポートフォリオサイトをメインとして運用します。
